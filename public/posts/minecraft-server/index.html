Minecraft Server

By Matthew

June 19, 2023

Minecraft servers are single threaded so having more CPU cores will not improve the server’s performance. Articles recommend 1GB to 2GB of allocated RAM for 10 people on a server.

Client Side

(1.20.1) Fabric is mod loader for Minecraft.

(1.20) OptiFine is an optimization mod for better performance. Usually you install OptiFine on its own without the Fabric mod loader but since we want voice chat we will need to supplement it with OptiFabric.

(1.19.4) OptiFabric allows OptiFine to run on the Fabric mod loader.

(1.20) Simple Voice Chat adds voice chat to Minecraft.

Server Side

(1.20.1) PaperMC is a Minecraft game server designed to improve performance.
